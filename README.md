# rubiks-cube-solver

A fully functional Rubik's cube solver with 3D visualization, step-by-step solving, and speedcubing tools.

## Features
- Interactive 3D cube (Three.js)
- Auto-solve with Kociemba algorithm
- Step-by-step solution visualization
- Scramble generator
- Built-in timer for speedcubers
- Photo input for cube state detection
- Offline support (PWA)
- Mobile responsive

## Setup
```bash
npm install
npm run dev
```

## Build for production
```bash
npm run build
```
